# Project Brief: TechOps 4.0 (TecniPro)

## Overview
Platform SaaS B2B for managing technical services in electronics, mechanics, and electricity. It connects companies (clients) with specialized technicians, managed by an administrative backoffice.

## Core Objectives
1.  **Centralized Management**: Unified platform for work orders, assets, and users.
2.  **Real-Time Operations**: Geolocation of technicians and live status updates.
3.  **Efficiency**: Automated dispatching (drag-and-drop), mobile access for technicians, and self-service for clients.
4.  **Scalability**: SaaS architecture designed to grow with multiple companies and technicians.

## Scope
*   **Web App (SPA)**: Public landing, Client Portal, Technician Portal, Admin Console.
*   **Backend**: RESTful API, Real-time services (WebSockets).
*   **Mobile**: PWA for technicians with offline capabilities.

## Key Metrics
*   Response time to service requests.
*   Technician utilization rate.
*   Client satisfaction (transparent tracking).
